# Runbook Skill

Use this skill for deployment, incident, and operational tasks.

A good runbook has:

- prerequisites
- exact checks to perform
- expected healthy signals
- decision points
- rollback or mitigation criteria
- owner handoffs
- post-incident follow-up

Avoid generic advice. Tie each step to an observable signal or an action.
