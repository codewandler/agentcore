# Source Notes Skill

Use this skill when summarizing or synthesizing notes.

Structure output as:

- Claims: statements supported by the provided material.
- Evidence: short references to the supplied text or repository context.
- Uncertainty: unresolved, contradictory, or missing information.
- Follow-up: the smallest useful next questions or sources.

Avoid overconfident language when the input is partial.
